## Usage
Install requirements:
```console
foo@bar:~$ pip3 install -r requirements.txt
```
Add a FB account and user id to the top of the code, and use that id in the message sender at the bottom.

Run the script:
```console
foo@bar:~$ python3 priceAlert.py
```
